# rfm-customer-segmentation

Customer segmentation using RFM analysis with Python and Pandas.

## Project Description
This project analyzes customer purchasing behavior using RFM (Recency, Frequency, Monetary) analysis and segments customers into groups.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Steps Performed
1. Data cleaning
2. Convert OrderDate to datetime
3. Calculate Recency, Frequency, Monetary values
4. Generate RFM scores
5. Segment customers (High Value, Loyal, Potential, At Risk)

## Output
